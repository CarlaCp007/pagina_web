Programación Estructurada

Fundamentos de la Programación Estructurada

La programación estructurada es un paradigma que organiza el código de manera clara y lógica, facilitando su comprensión y mantenimiento. Se basa en tres principios fundamentales:
Secuencialidad 📜: Las instrucciones se ejecutan en el orden en que aparecen.
Selección (Decisión) 🔀: Permite ejecutar diferentes bloques de código según ciertas condiciones (if-else, switch).
Iteración (Repetición) 🔁: Utiliza bucles (for, while) para repetir instrucciones hasta cumplir una condición.
Este enfoque evita el uso excesivo de saltos (goto), promoviendo la legibilidad y modularidad del código mediante funciones y estructuras bien definidas.
Es la base de la programación moderna y permite desarrollar software eficiente y organizado.

Variable y tipos de Datos

Las variables son espacios en la memoria de la computadora donde se almacenan valores que pueden cambiar durante la ejecución de un programa. Son fundamentales, ya que permiten guardar y manipular información de manera eficiente y lógica dentro de un programa.
Los tipos de datos determinan qué tipo de valores puede contener una variable. Los más comunes son:
Enteros (int): Números sin decimales (ej. 10, -5, 42).
Flotantes (float): Números con decimales (ej. 3.14, -0.5).
Caracteres (char): Un solo símbolo o letra (ej. 'A', '9', '?').
Cadenas de texto(string): Texto compuesto por varios caracteres (ej. "Hola mundo").
Booleanos (bool): Representan valores de verdad (true o false).

#Caracteres (char)

# Cadenas de texto (str)

# Booleanos (bool)

Operadores y Expresiones en Programación

En programación, los operadores son símbolos que permiten realizar cálculos y comparaciones entre valores. Una expresión es una combinación de variables, valores y operadores que se evalúan para obtener un resultado.
A continuación, exploramos los tipos principales de operadores.

1️⃣ Operadores Aritméticos 🧮

Se utilizan para realizar operaciones matemáticas básicas.

Operador Descripción Ejemplo
+ Suma a + b
− Resta a - b
* Multiplicación a * b
/ División a / b
% Módulo (resto de la división) a % b

2️⃣ Operadores Relacionales 🔍

Se usan para comparar valores y devuelven true o false.

Operador Descripción Ejemplo (a=5, b3) Resultado
== Igual a a==b false
!= Diferente de a != b true
> Mayor que a > b true
< Menor que a < b false
>= Mayor o Igual que a >= b true
<= Menor o Igual que a <= b false

3️⃣ Operadores Lógicos ⚡

Se usan para combinar expresiones condicionales.

Operador Descripción Ejemplo Resultado
AND Ambas condiciones deben ser true (a > 3) && (b < 5) True
OR Una de las condiciones deben ser true ` True

Estructuras de Control de Flujo

Las estructuras de control de flujo permiten modificar la secuencia en la que se ejecutan las instrucciones de un programa. Se dividen en estructuras condicionales (decisión) y estructuras repetitivas (bucles o iteraciones).

Estructuras Condicionales (Decisión)


Condiciones simples y múltiples.- Se usan cuando se necesita evaluar una condición y ejecutar instrucciones según el resultado.
If es el condicional principal con la cual se da inicio al bucle. Si deseamos incluir otras condicionales utilizaremos, dependiendo el lenguaje de programación, las siguientes variables else-if, elif. En el momento en que una condición se evalúe como true , el resto de condiciones serán ignoradas. En el caso de que ninguna condicional se cumpla encontramos la condicional else.

Ejemplo de Código 

#include 

int main() {
    int nota;
    printf("Ingresa tu calificación: ");
    scanf("%d", & nota);

    if (nota >= 90) {
        printf("Excelente\n");
    } else if (nota >= 70) {
        printf("Aprobado\n");
    } else {
        printf("Reprobado\n");
    }
    return 0;
}
Ejemplo de Código 

#include

Estructuras Repetitivas (Bucles o Iteraciones) 🔁

Permiten ejecutar un bloque de código varias veces mientras se cumpla una condición. Si es falsa, el bucle termina.


WhileRepetición con condición inicial.- Verifica la condición antes de cada iteración y solo se ejecuta mientras la condición sea true.
Do-While Repetición con ejecución garantizada.- Ejecuta el bloque al menos una vez, aunque la condición sea falsa desde el inicio.

Ejemplo de Código 

#include 

int main() {
    int contador = 1;
    while (contador <= 5) {
// Se ejecuta mientras contador sea 
// menor o igual a 5
       printf("Iteración %d\n", contador);
       contador++;
// Incrementa contador en 1
    }

    return 0;
}
Ejemplo de Código 

#include 

int main() {
    for (int i = 1; i <= 10; i++) {
        if (i == 5) {
            printf("Se detiene en 5\n");
            break;  // Sale del bucle
        }
        printf("%d\n", i);
    }
    return 0;
}


For Bucle con contador definido.- Se usa cuando se conoce el número exacto de iteraciones.
Control de Bucles: Break y Continue:
Break Detiene el bucle antes de que se cumpla la condición.
Continue Salta una iteración y pasa a la siguiente.
